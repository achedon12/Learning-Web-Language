Vous allez apprendre comment faire des listes en html

Informations

Il existe 2 types de listes en html : les listes ordonnées (voir exemple ci-dessous) et les listes désordonnées ( voir exemple ci-dessous). Pour réaliser des listes, il ne faut pas oublier de mettre la balise "li" qui permet d'intégrer des éléments de listes

Exemple de liste ordonnée

  1. texte1
  2. texte2
  3. texte3

Exemple de liste désordonnée

  • texte1
  • texte2
  • texte3
