Apprendre le html - Faire des paragraphes

Vous allez apprendre comment faire des paragraphes en html

Informations

La balise d'un paragraphe est "p". Elle place une ligne vide au-dessus et en dessous du texte pour le mettre en évidence, et le navigateur la considerera comme telle. Le paragraphe est un élément de base en traitement de texte.
